# Тест-кейсы для микросервиса объявлений

## 1. Создание объявления (POST /api/1/item)

### 1.1 Позитивные тесты
- **TC-001**: Создание объявления с корректными данными
- **TC-002**: Создание объявления с минимальной ценой
- **TC-003**: Создание объявления с максимальной ценой
- **TC-024**: Создание объявления с названием из специальных символов

### 1.2 Негативные тесты - обязательные поля
- **TC-004**: Создание объявления без sellerID
- **TC-005**: Создание объявления без названия
- **TC-006**: Создание объявления без цены
- **TC-010**: Создание объявления без статистики

### 1.3 Негативные тесты - невалидные значения цены
- **TC-007**: Создание объявления с отрицательной ценой
- **TC-008**: Создание объявления с ценой больше максимальной
- **TC-009**: Создание объявления с нулем в цене

### 1.4 Негативные тесты - невалидная статистика
- **TC-011**: Создание объявления без лайков
- **TC-012**: Создание объявления без количества просмотров
- **TC-013**: Создание объявления без контактов
- **TC-014**: Создание объявления с отрицательным числом в количестве лайков
- **TC-015**: Создание объявления с нулем в количестве лайков
- **TC-016**: Создание объявления с отрицательным числом в количестве просмотров
- **TC-017**: Создание объявления с нулем в количестве просмотров
- **TC-018**: Создание объявления с отрицательным числом в контактах
- **TC-019**: Создание объявления с нулем в контактах
- **TC-020**: Создание объявления с числом выше позволенного в лайках
- **TC-021**: Создание объявления с числом выше позволенного в просмотрах
- **TC-022**: Создание объявления с числом выше позволенного в контактах

### 1.5 Негативные тесты - валидация названия
- **TC-023**: Создание объявления с пустым названием
- **TC-025**: Создание объявления с названием из пробелов

### 1.6 Негативные тесты - валидация sellerID
- **TC-026**: Создание объявления с sellerID за верхней границей
- **TC-027**: Создание объявления с sellerID за нижней границей
